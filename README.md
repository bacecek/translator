# Переводчик
Тестовое задание для Школы мобильной разработки "Мобилизация" от Яндекса в 2017 г.
Стек технологий:
- Yandex Translator API & Dictionary API;
- Yandex SpeechKit;
- Moxy;
- Dagger 2;
- Realm;
- Retrofit;
- RxJava;
- Butterknife;
- EventBus;
